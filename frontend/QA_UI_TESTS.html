7. Package Lifecycle Tests

Critical 16 Test Cases
Package Status Flow:
NEW → OPEN_FOR_BIDS → BID_SELECTED → PENDING_PICKUP → IN_TRANSIT → DELIVERED
CANCELED can occur from any non-terminal state (except IN_TRANSIT).
FAILED → OPEN_FOR_BIDS (admin only retry)

See full test cases in the markdown version of this document.

8. Bidding System Tests

High 16 Test Cases
Bid Timeline:
• First bid starts 24-hour deadline for sender to select
• If deadline expires: auto-extends 12 hours (max 2 times)
• After 2 extensions: all bids expire, package returns to OPEN_FOR_BIDS
• 6 hours before deadline: warning notification sent

See full test cases in the markdown version of this document.
